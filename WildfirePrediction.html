Wildfire Prediction and Education App

An app built on Azure using Machine Learning to analyze weather and wildfire data,
predicting where wildfires will strike next.

I worked on a team in collaboration with a Microsoft mentor to build a Wildfire Prediction and Education application. I was orignally tasked with creating the database and processing input data, but quickly spread to fill multiple roles in the project as needed to fill gaps.
In the end, I created the database, the framework of the website and the map display, and helped with getting the Machine Learning segment of the project running. This helped me learn how to be flexible on projects and shift what role I fill as needed by the team, as well as how to coordinate with team members to manage issues as they come up.
